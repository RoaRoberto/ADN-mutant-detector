# Vista del Detector de Mutantes

La aplicación consta de una **Vista Principal** única, diseñada para ser intuitiva y directa:

1. **Panel de Entrada**: Un espacio donde puedes escribir o pegar las secuencias de ADN.
2. **Visualización de la Matriz**: Una cuadrícula que organiza visualmente las letras del ADN para que puedas ver las secuencias fácilmente.
3. **Botón de Acción**: Un botón claro para iniciar el análisis.
4. **Panel de Resultado**: Un área que te dirá si el ADN analizado pertenece a un humano o a un mutante, usando colores para que sea fácil de distinguir.
